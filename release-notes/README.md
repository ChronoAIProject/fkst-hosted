# release-notes/

Per-release notes for fkst-hosted. Each release adds one dated file here:

```
release-notes/release-note-YYYYMMDD-HHMM.md
```

copied from [`.github/release-note-template.md`](../.github/release-note-template.md)
and filled in with the user-facing notes for that release.

These files are **ephemeral**: after a release is tagged, the dated file is
removed by the automated post-release cleanup. The permanent record of every
release lives in the root [`CHANGELOG.md`](../CHANGELOG.md).
