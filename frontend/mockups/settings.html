Suspend deployment
A hosted-service lifecycle action on ChronoAI cloud (not an engine-internal API) — POST /api/v1/sessions/:id/stop (202 accepted; the console polls GET /sessions/:id through stopping → stopped). No pipelines or packages run and no polls fire while stopped. The durable ledger is preserved; resume = POST /sessions for the package. Full stop.
Delete account
Removes this account and the NyxID-brokered credentials it holds (GitHub access, hosted backend). Connected repositories are released and the console can no longer reach GitHub or the engine. Irreversible.
